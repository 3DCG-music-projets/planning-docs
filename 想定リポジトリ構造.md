# 想定リポジトリ構造

LifeStreamStudioはOrganazationの名前である。

```
LifeStreamStudio/
├── assets-3dmodels/          # モデル制作（Blenderファイル等）
├── assets-animation/         # アニメーションファイル（FBXなど）
├── unity-project/            # Unityプロジェクト本体
├── planning-docs/            # 台本・絵コンテ・構成など（Notion連携可）
├── music-sound/              # BGM・SE素材など
├── export-videos/            # 出力された映像や成果物
├── tools-scripts/            # 自動化や補助スクリプト（例：Blender API、Unity C#）
```
